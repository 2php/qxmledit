QXmlEdit

Simple XML editor.

Download this project as a .zip file Download this project as a tar.gz file

QXmlEdit is a simple XML editor written in qt4. Its main features are unusual data visualization modes, nice XML manipulation and presentation and it is multi platform. It can split very big XML files into fragments, and compare XML files. It is one of the few graphical Open Source XSD viewers.

Notably QXmlEdit can split very huge XML files with a set of configurable parameters and output options as shown in this tutorial.

Latest release is 0.8.14, and its main new feature is a data anonymization tool.

Main features:

  • Hierarchical customizable view of XML elements.
  • Fast XML hierarchy navigation.
  • Split of big XML files.
  • Search supporting XPath expressions.
  • Base 64 data handling.
  • Custom visualization styles.
  • XML Schema (XSD) viewer.
  • Columnar view.
  • Sessions handling.
  • Graphical XML file view.
  • Map view of a XML document.
  • Split and fragment extraction of big XML files.
  • Visual compare of XML Schema files.
  • Visual compare of XML files.
  • XML Snippets.
  • XSL specialized mode.
  • Data anonymization.

Requirements

QT libraries version 4.8 or higher are required. QT 5 is supported.
